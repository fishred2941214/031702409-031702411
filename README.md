**使用说明**
- 请将**家族树**文件夹下载或是将上述的文件统一下载并放入同一文件夹中，以防无法完整显示。
- 使用浏览器打开**tree.html**文件，即可在浏览器中看到页面。
- 在左侧的文本框输入数据，点击**确定**按钮，将会在右侧生成一棵以导师为根节点的树。
- 单击小三角或是双击节点内容如“张三”可展开或收起节点。
- 在节点处右键单击，可进行修改、删除、添加节点的操作。
   - 修改节点：对节点进行重命名。
   - 删除节点：删除该节点及其下级节点。
   - 添加节点：在该节点处新增一个下级节点。
